Comprehensive Curriculum

Month 1-2: IT Fundamentals

8 weeks
  • Computer Hardware Components & Assembly
  • Operating Systems (Windows, Linux basics)
  • Troubleshooting Methodologies
  • Basic Command Line Operations
  • PC Maintenance and Diagnostics

Month 3-4: Networking & Cloud

8 weeks
  • Network Fundamentals (TCP/IP, OSI Model)
  • DHCP, DNS, and Network Services
  • Router and Switch Configuration
  • Microsoft 365 Administration
  • Cloud Computing Basics (Azure, OneDrive)

Month 5-6: Security & Administration

8 weeks
  • Cybersecurity Fundamentals
  • Active Directory Management
  • User Account Administration
  • Group Policy Implementation
  • Data Backup and Recovery

Month 7: Professional Skills

4 weeks
  • Help Desk and Ticketing Systems
  • ITIL Framework and Best Practices
  • Remote Support Tools (RDP, TeamViewer)
  • Customer Service Excellence
  • Resume Building and Interview Prep
